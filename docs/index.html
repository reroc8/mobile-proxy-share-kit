Shadowrocket AI 风控稳定规则

核心目标只有一个：让 Claude、ChatGPT、Gemini 等 AI 服务走清晰、稳定的策略组，不和普通 Google / YouTube / Proxy 混在一起。

打开完整骨架模板 高级用户：只打开规则片段 查看 GitHub 项目

模板链接：https://raw.githubusercontent.com/reroc8/mobile-proxy-share-kit/main/shadowrocket/Shadowrocket.full.conf

先看这一句

这不是节点订阅，不含任何节点。你必须先在 Shadowrocket 导入自己的订阅，再使用这份模板或规则。

这也不是去广告规则。基础版不启用广告拦截，优先减少登录、验证码、支付、AI 风控接口误伤。

三步使用

  1. 先在 Shadowrocket 导入自己的订阅。
  2. 导入本页面的完整骨架模板，然后断开并重新连接 Shadowrocket。
  3. 进入策略组，把自己的节点放到 Claude、AI、Exchange 等对应位置。

二维码指向完整骨架模板，不包含任何节点。导入后还要使用自己的订阅节点。
